Seletores CSS

CSS é utilizado para estilizar páginas web. Ele permite adicionar cores, fontes e layouts às páginas HTML.

Seletores Utilizado? Comentário
Seletor de tipo Seleciona elementos com base no seu tipo (tag). Ainda não utilizei este seletor para aplicar estilos aos parágrafos em um exemplo de página.
Seletor de classe Seleciona elementos com base na sua classe. Ainda não utilizei este seletor em nenhuma página.
Seletor de ID Seleciona um elemento com base no seu ID único. Ainda não utilizei este seletor para estilizar o cabeçalho de uma outra página.

Combinadores de Seletores

Combinador Utilizado? Comentário
Descendente Seleciona elementos que são descendentes diretos ou indiretos de outro elemento. Ainda não utilizei este combinador em um projeto para estilizar listas dentro de divs.
Filho direto Ainda não utilizei este combinador em nenhuma página.
